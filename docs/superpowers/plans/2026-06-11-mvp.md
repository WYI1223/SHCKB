# SHCKB MVP Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Local single-user SHCKB instance proving create → edit → publish → read, on non-disposable product-shaped architecture.

**Architecture:** Bun-workspaces monorepo. Pure-function `packages/grid-engine` shared by client (interaction) and server (pre-persist validation). `apps/server` = Hono + Drizzle + SQLite with PEP-stub middleware, two-state notepage model (normalized working state + published JSON snapshot). `apps/web` = React + Vite SPA; grid editor lifted from carryover prototype; markdown as the only `BlockKindModule` behind a static registry.

**Tech Stack:** Bun 1.3 / Hono / Drizzle + bun:sqlite / React 18 + Vite / react-markdown + remark-gfm + rehype-sanitize / vitest. Per [ADR-0019](../../engineering/decisions/ADR-0019-mvp-implementation-baseline.md), scope per [mvp-scope-2026-06-11](../../engineering/design/discussions/mvp-scope-2026-06-11.md).

## Locked contracts (decisions; implementation must conform)

### C1 — Engine target contract changes (vs carryover)

- `type BlockKind = string` (open identifier; delete closed union).
- `DEFAULT_SIZES` and any kind→size lookup **leave the engine**. New signature:
  `inferDropIntent(state, cursorCol, cursorRow, size: { colSpan: number; rowSpan: number }): DropIntent`
  (caller passes the block-kind module's defaultSize).
- `validateState(state, opts?: { gravity?: boolean })` — gravity-stability check only when `opts.gravity !== false` (gravity toggle D7: gravity-off pages persist non-stable layouts legally).
- Everything else (ops, OpResult, collision, occupancy, gravity) lifts unchanged.
- CONTRACT.md rewritten: remove ProseMirror/editor-shell/“future ADR-0019-grid-engine” residue; document the two changes above; invariant wording “always gravity-stable” → “gravity-stable when gravity enabled”.

### C2 — DB schema (Drizzle, SQLite)

```ts
// apps/server/src/db/schema.ts
import { sqliteTable, text, integer } from 'drizzle-orm/sqlite-core';

export const notepages = sqliteTable('notepages', {
  id: text('id').primaryKey(),                    // nanoid
  slug: text('slug').notNull().unique(),          // from title + suffix on collision
  title: text('title').notNull().default('Untitled'),
  visibility: text('visibility', { enum: ['private', 'public'] }).notNull().default('private'),
  gravityEnabled: integer('gravity_enabled', { mode: 'boolean' }).notNull().default(true),
  publishedDoc: text('published_doc'),            // JSON snapshot | null
  createdAt: integer('created_at', { mode: 'timestamp_ms' }).notNull(),
  updatedAt: integer('updated_at', { mode: 'timestamp_ms' }).notNull(),
});

export const blocks = sqliteTable('blocks', {     // working state
  id: text('id').primaryKey(),
  notepageId: text('notepage_id').notNull().references(() => notepages.id, { onDelete: 'cascade' }),
  kind: text('kind').notNull(),
  col: integer('col').notNull(),
  row: integer('row').notNull(),
  colSpan: integer('col_span').notNull(),
  rowSpan: integer('row_span').notNull(),
  content: text('content').notNull(),             // JSON, kind-owned shape; markdown = {"markdown": string}
});
```

`publishedDoc` JSON shape: `{ title, gravityEnabled, blocks: [{id, kind, col, row, colSpan, rowSpan, content}], publishedAt }`.

### C3 — API surface (Hono, all JSON)

| Method + path | Auth | Behavior |
|---|---|---|
| `GET /api/notepages` | PEP | list `{id, slug, title, visibility, hasPublished, updatedAt}` |
| `POST /api/notepages` | PEP | body `{title?}`; create private page; generate slug; 201 `{id, slug}` |
| `GET /api/notepages/:id` | PEP | `{page, blocks}` working state |
| `PUT /api/notepages/:id/working-state` | PEP | body `{title, gravityEnabled, blocks: BlockWithContent[]}`; server builds GridState, runs `validateState(state, {gravity: gravityEnabled})`; **422 + error list** on violation; else transactional replace of blocks + page fields |
| `POST /api/notepages/:id/publish` | PEP | snapshot current working state → `publishedDoc`; 200 `{publishedAt}` |
| `POST /api/notepages/:id/visibility` | PEP | body `{visibility: 'private'\|'public'}` |
| `DELETE /api/notepages/:id` | PEP | hard delete (MVP; tombstones out of scope) |
| `GET /api/public/notes/:slug` | none | 404 unless `visibility='public' && publishedDoc != null`; returns publishedDoc. Privacy: 404 (not 403) for private/missing alike per [notepage.md] no-leak |

PEP middleware: sets `ctx.user = Object.freeze({ id: 'local-author', role: 'author' })` on `/api/*` except `/api/public/*`. No identity machinery (D3).

### C4 — BlockKindModule seam (web)

```ts
// apps/web/src/blocks/types.ts
export type BlockViewProps<C = unknown> = {
  content: C;
  onChange: (next: C) => void;   // updates author working state (host-orchestrated)
};
export type BlockKindModule<C = unknown> = {
  kind: string;
  label: string;
  glyph: string;
  defaultSize: { colSpan: number; rowSpan: number };
  createContent: () => C;
  EditView: React.ComponentType<BlockViewProps<C>>;     // mounted only on active block
  RenderView: React.ComponentType<{ content: C }>;      // inactive preview + read route
  extractText: (content: C) => string;                  // AST-derived, not DOM scraping
};
// apps/web/src/blocks/registry.ts — static object, no machinery:
export const BLOCK_KINDS: Record<string, BlockKindModule> = { markdown: markdownModule };
```

Markdown module: react-markdown + remark-gfm + rehype-sanitize behind `blocks/markdown/`; EditView = textarea + live preview (source-plus-preview); extractText via mdast plain-text walk. Package names are spike-level ([block-markdown.md]); render/extract stay behind this module boundary.
