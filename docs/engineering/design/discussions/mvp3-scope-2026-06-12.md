## 决策摘录（详见 spec）

- M3-D1 格式即契约：canonical 目录树（manifest + tree/ + blobs/），确定性导出
- M3-D2 格式迁移管线：镜像 [ADR-0020] 模式的纯函数管线；双向；降级在导出端；有损必须显式；up/down 成对纪律
- M3-D3 Import 仅空实例全量恢复；原子性；新格式进旧实例明确拒绝
- M3-D4 Blob 引用枚举为 export/GC 共享逻辑；块 kind 必须以小写 hex sha256 原文引用 blob（新契约，入 ADR-0023）
- M3-D5 不做：双向同步 / 增量 import / publishedHtml 导出

## Build log

1. **BlobStore 扩展**：read/list/delete 三方法（store 保持 policy-free，引用判断归调用方）。
2. **引用枚举**（blob-refs.ts）：kind-opaque 扫描任意 JSON 中 64-hex 字符串；契约 = kind 以小写 hex sha256 原文引用 blob（M3-D4）。误报方向安全：export 与 blobs 表求交，GC 误报只多保活。
3. **格式模块**（format.ts）：FORMAT_VERSION=1、canonicalJson（2-space + LF）、sanitizeDirName（路径敌意字符 + Windows 保留尾缀）。测试期望值修过一处下划线计数错误。
4. **Exporter**：确定性铁律落地——同级排序 (sortKey, id)、blocks 按 id、manifest 字典序、目录碰撞大小写不敏感加 ~n 后缀。确定性测试：两次导出仅 exportedAt 异。
5. **格式迁移管线**（migrate-format.ts）：up/down 成对、降级损失显式枚举、缺级 throw。production registry 空，机器由合成 v1↔v2 transform 对测透（升级/降级/无损往返/缺路径 5 例）。
6. **Importer**：四道门全在写入前（空实例 409 / 新格式 409 带降级提示 / 结构+逐页 validateState+blob sha256 复核 422 / 原子单事务）。blob 文件先写（幂等，孤儿归 GC）。**Round-trip 字节级一致测试一次通过**。
7. **Admin 路由**：export zip（fflate，mtime 固定 DOS epoch——`new Date(0)` 越界 zip 时间戳下限 1980，实测抓到）/ import zip / blobs/gc。requireAdmin 是第一个角色差异授权点；author 角色 403 用 bootstrap 临时实例技巧造第二用户验证。
8. **Web**：侧栏 admin-only ⤓ Export 锚点（cookie 直接随行，无需 JS 下载逻辑）。
9. **文档**：ADR-0023 + README 索引 + runbook 逻辑备份节 + AUDIT 新增 MVP-era debts 小节（GC 债标记 repaid，SSR/SPA drift 挂账）。
10. **工程备忘**：本轮两次在生成文件里混入字面 U+0000/U+001F 控制字节（regex `U+0000-U+001F` 转义被序列化为真实字符），byte-level 检查后修复——以后写含控制符转义的代码用十六进制转义并立即校验文件字节。
11. **全量验证**：engine 44 / server 68 / web 5 全绿；server+web typecheck 干净。
12. **容器 round-trip 实测**（compose.dev 全新卷）：API 造数（文件夹+页+image blob+publish+public）→ export zip（1417B）→ down -v 全新实例 → import `{folders:1,pages:1,blocks:2,blobs:1}` → 静态页 200 / blob 字节一致 / 公开树完整 → **再导出与原导出除 manifest.exportedAt 外零差异**。
13. **Dev 链路验证**：owner 真 dev 库经 vite 代理导出 103KB，方法论/笔记法 中文目录名原样镜像；侧栏 admin ⤓ Export 入口 Playwright 确认渲染。坑：重启 dev server 时 CWD 在 apps/server 导致相对路径 data/dev.db 新建空库（已删，owner 数据无损）——dev server 必须从仓库根启动。
