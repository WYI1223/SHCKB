Owner 愿景（2026-06-12 对话，原话要点）：右键背景可改背景颜色/铺图片；右键 block 可改它现在的样子（纸质换白卡、调纹路、隐藏 image 底纸只显示图）；未来 richtext 选中文字时自己调用这个面板声明可改项（字体/颜色/大小/间距）；或者像 3D 引擎侧栏——点一个 item 显示它的 properties。Owner 确认该物通常叫 **properties**。同轮裁定：richtext 不急、主题（operator 级）调整到此为止。

## 决策摘录（详见 spec）

- **M6-D1 模型 = selection → contributors → property sections**：选择类型 page | block |（预留 text-range）；贡献者三类 = host（页面级属性）、theme（外壳策展选项）、kind module（内容工具，MVP-5 已有）。现 ToolPanel 是该模型的退化形态，升级为 Properties 面板。
- **M6-D2 形态先停靠后悬浮**：检查器停靠于侧栏目录下方（owner 的 3D 引擎心智模型），右键悬浮菜单 = 同一属性区块的快捷投影，后做。形态 = dev 决策，内核（属性区块模型）才是产品锁。
- **M6-D3 作者级外观 = 第三自定义轴**，纪律与 M5-D3 同构：**theme 策展块壳选项，作者在选项内挑**（不是自由改壳——否则从后门绕回自由调色）。`Theme.blockShellOptions?` 声明 per-kind 可选外壳变体与开关；块存 `shellChoice`（presentation 数据，不进 kind content）。
- **M6-D4 页面背景**：页面级属性（背景色 / blob 图片引用），同样 theme 可策展可选背景集；进 publish 快照渲染。
- **M6-D5 存储与格式**：blocks/page 级新外观字段 → schema migration + **format v4 成对 transform**（down 丢外观逐项报损失）。
- **M6-D6 还 ADR-0026 挂账**：stationery 槽位组件改"渲染时读参数"（块壳选项的前提），顺带解锁 stationery palettes。
- **M6-D7 richtext 预留**：selection 模型含 text-range 类型占位 + 贡献接口形状；不实现。
