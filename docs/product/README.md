# Product

产品视角文档。WHY this product exists + WHAT it does + 已发生的产品决策 retrospective。

## 子目录

- [`prd/`](./prd/) —— 项目级 PRD + 功能级 PRD（living；持续更新）
- [`retrospectives/`](./retrospectives/) —— 已发生的反思 / cognitive defaults / framing critique（写完不改）

## 何时读这里

- **新 contributor onboarding** → 先看 [`prd/project.md`](./prd/project.md)
- **新 feature 立项 / 评估** → 看相邻 feature 的 PRD 体例 + [`prd/features/README.md`](./prd/features/README.md)
- **决策失败复盘** → 看 [`retrospectives/`](./retrospectives/)

## 何时不读

- 想知道"DB 用什么"/"协议怎么选" → 去 [`../engineering/decisions/`](../engineering/decisions/)
- 想知道"系统怎么 compose 起来" → 去 [`../engineering/design/architecture-overview.md`](../engineering/design/architecture-overview.md)
