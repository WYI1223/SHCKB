测量管线 — 量内容，不量壳

autofit 要知道内容到底多高。关键是测量必须与壳无关（frame-agnostic）——否则一个绝对定位的主题就能把它骗成 0。

✗ 前 · 量 frame 外层 auto 高
.skb-paper { position: absolute; inset: 3px }
外层 offsetHeight → 塌成 0
（绝对定位脱离了正常流）
fit = 1 ← 6 行内容压进 1 行

壳一变，测量就崩。stationery 量到 0，galley 量到 6——同样内容，两种结果。

✓ 后 · 离屏定高探针 · 量 AREA / CONTENT
MEASURE_CELL_PX（定高）
AREA：height: 100% → 得「chrome + area」
CONTENT：height: auto → 得内容自然高
chrome = MEASURE_CELL_PX − area
fit = ⌈( content + chrome + 2·pad ) / slot ⌉

定高离屏 + 两次相对测量 → 反推出 chrome 与内容高，与任何壳无关。stationery / galley 现在量出同一个数。
