Navigate · resolveTarget(surface, ref)

保面导航 — 按当前面解析同一个 LinkRef

点一个内部链接 = 解析 （当前 surface, LinkRef） → 一个动作。客户端完成，Shell 不卸载；同页 block 目标根本不导航——纯滚动。

scroll edit view read note
resolveTarget( pathname , LinkRef { pageId, blockId? } )
▼
同页 + 有 blockId
→
scroll — scrollToBlock，URL 不变（不导航）
edit 面
→
navigate /edit/:pageId(#blockId)
view 面
→
navigate /view/:pageId(#blockId)
read 面
→
navigate /read/:pageId (留整库浏览，不逃 note)
note 面
→
navigate /notes/:pageId (留单页 bare)
客户端：router push，不整页重载
外科手术式：Shell 布局常驻
面守恒：edit→edit，read→read
全 id 后解析器塌成「同面 + 同 id」—— /read/ → /notes/ 逃逸结构性消失
