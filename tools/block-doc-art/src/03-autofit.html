Autofit · follow / fix

autofit — 跟随内容，还是固定高度

block 顶端能力，逐 kind 默认。 两个模式，二选一：follow（默认）高度跟随内容；fix 固定高度、内容滚动。无 floor、无 minRowSpan —— 跟随就是跟随，固定就是固定。

follow 默认
文本 kind（markdown / richtext / code）的默认
高 = 内容
  • 高度 = 实测内容 fit（1 行下限，在 measureFit）
  • 打字撑高 · 删回缩回，都跟随内容
  • overflow 裁剪（rowSpan == fit，刚好填满）
  • 无竖向 resize 手柄（高度归内容）
  • 可逆 C5：从不可变 base 快照重算 → 精确归位
fix opt-in
image 的唯一模式（fix-only · canFollow:false）
高 = 你设定
  • 固定手动 rowSpan（可拖拽改）
  • 内容超出 → 滚动（overflow: auto）
  • 切入时冻结当前显示高（follow→fix 不跳变）
  • 暴露竖向 resize 手柄
☑ Fixed height 右键菜单的 opt-in 勾选框 —— 勾选 = fix，默认 follow 不勾。
逐 kind 默认： markdown → follow richtext → follow code → follow image → fix-only
fit = ⌈(内容高 + chrome + 2·pad) / slot⌉
