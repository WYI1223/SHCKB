autofit — 有效高 = max(floor, fit)，且可逆

这是 block 顶端能力，不是 markdown 专属。 作者定下限（floor），内容实测撑高（fit），整行往上长、不回退；删回又精确归位。

= minRowSpan
（作者最小高）
fit
= 实测行数
（仅编辑时测）

有效 = max(floor=3, fit=6) = 6

不持久 · 不写库
打字 → 撑高一行变多行，块整行往上长……
⇄可逆
删回 → 归位删到比 floor 矮，精确回到 floor，不残留。
可逆（C5）：每次手势都从不可变 base 快照重算 effective，所以撑高与归位都精确、无漂移。测量发生在离屏，永不写库。
