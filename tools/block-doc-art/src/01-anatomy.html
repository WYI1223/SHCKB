Block anatomy · ADR-0029

一个块的解剖 — 从外到内，缩进即包含

功能归 host；kind / theme / UI 是薄插件，在同一个 host 盒上组合 — 无基类。 每一圈被外圈包住；星标那圈 .skb-content-box 是 host 拥有的可测量内核。

host 拥有 theme · skin（材质） kind · 内容 canvas · 几何 UI-plugin（延后）
① canvas · 几何定位
② .skb-frame-root front(ctx) · 和纸胶带 skin.root · tilt / filter
behind(ctx) · 盒后装饰层（在内容盒之前绘制）
③ .skb-content-box ★ host 拥有 skin.box · 仅视觉 Pick<>
④ RenderView
kind 的内容 markdown · richtext · image · code
host 最后落 · position · width · height · overflow ✓ — skin 永远赢不了
①
canvas · 几何host 给 left / top / width / height，外加拖拽 / 缩放手柄与选中光环。编辑器拥有外层定位。
②
.skb-frame-roothost 渲染的元素。skin 只供 root.style（tilt / filter 等纯视觉）与 behind / front 装饰，碰不到结构。
③
.skb-content-box ★host 拥有：in-flow · 撑出自然高 · 持 overflow。skin.box 视觉先铺，host 把 position/size/overflow 最后落。
④
RenderViewkind 只供内容（md / rich / image / code），碰不到盒。第三方 kind 同此一条路。
